Метод для получения последнего элемента из списка работает за *O(1)*
```c++

{
Item *item = new Item(x);
if (last != nullptr)
	last->next = item;
last = item;
if (first == nullptr)
	first = last;
    size++;
}
```

###Для списка:
```c++
void pop_head() 
    {
    if (first == nullptr)
      return;
    Item *t = first;
    first = t->next;
    delete t;
    size--;
    if (first == nullptr)
      last = nullptr;
    size--;
    }
```
Метод для удаления первого элемента из списка, работает за *O(1)*
###Для вектора:
```c++
void pop_head(){
```

#Задание 5(Г)
